Exact text/regex search over the repository's working tree (like ripgrep). Unlike query (semantic, ranked by meaning), this finds the LITERAL pattern and returns matching lines as `path:line: text`, plus addressable chunk indices you can pass to add_chunks. Use it to find every call site of a symbol, where a constant is defined, or all uses of an identifier. Does NOT consume the query turn budget. Respects .gitignore; skips binary files.
